# 证据驱动市场细分方法论（速查）

> 本文件是 `drug-market` Skill 的 references 之一，供簇C「证据驱动市场细分」引用。
> **本文件是 drug-market Skill 的核心创新**——用循证证据强度分布驱动市场细分，取代传统拍脑袋细分。
> last-reviewed: 2026-07-10

## 1. 证据强度分布图（核心方法）

### 1.1 主线逻辑
```
循证证据强度分布 → 锚点人群 → 细分优先级 → 推广信息与资源倾斜
```
细分不是拍脑袋，是从医学证据里长出来的。产品在哪个亚组证据最强，那个就是首批推广的锚点人群。

### 1.2 三维交叉建表

按 **适应症 × 亚组 × 关键终点** 三维交叉建表，**每条循证必须带原文名称**（研究名/指南全称/共识全称），禁止只写"XX推荐""XX证实"等模糊表述：

| 适应症 | 亚组 | 关键终点 | 研究原文 | 证据等级 | 样本量N= | 指南推荐 | 专家共识 | 关键结果 | 锚点判定 |
|---|---|---|---|---|---|---|---|---|---|
| 2型糖尿病 | 心血管高危 | MACE(3P) | EMPA-REG OUTCOME(NEJM 2015)🔍 | 1a(Meta) | 10,142 | ESC 2023 CVD指南 Ⅰ/A | 《中国2型糖尿病防治指南》(中华医学会糖尿病学分会, 2024) 一线推荐 | HR 0.86 | ⭐锚点 |
| 2型糖尿病 | 伴CKD | 肾脏复合终点 | CREDENCE(NEJM 2019)🔍 | 1b(RCT) | 12,874 | KDIGO 2024 CKD指南 Ⅰ/A | 《糖尿病肾脏疾病临床诊疗中国指南》(2023) 推荐 | HR 0.78 | ⭐锚点 |
| 2型糖尿病 | 普通人群 | HbA1c降幅 | 多中心RCT🔍 | 1b(RCT) | 8,058 | ADA 2024糖尿病诊疗标准 A级 | — | -1.4% | 二批 |
| 减重 | BMI≥30 | 体重降幅% | STEP 1(NEJM 2021)🔍 | 1b(RCT) | 1,961 | — | 《中国超重/肥胖医学营养治疗指南》(2021) 二线推荐 | -15% | 二批 |

### 1.3 标注标准（五维 + 研究原文名称，共六项）

每格必须标注以下六项（调用 `references/evidence-levels.md`）。**严禁只写"XX推荐""XX证实"等模糊表述，每条循证必须带原文名称：**

| 标注维度 | 标准 | 来源 | 反例（禁止） |
|---|---|---|---|
| **研究原文** | 研究全名（缩写）+ NCT号 + 发表期刊+年份（如 HARMONIZE, NCT02088073, JAMA 2014） | ClinicalTrials.gov / PubMed | "有RCT支持" ← 不知道是什么研究 |
| 证据等级 | 1a(Meta)/1b(RCT)/2a(队列)/2b(病例对照)/3(病例系列)/4(专家意见) | evidence-levels.md §1 | "证据强" ← 没有分级 |
| 样本量 N= | 关键研究的样本量；Meta注明纳入研究数 | 研究原文 | — |
| 指南推荐级别 | 指南全称 + 机构 + 年份 + 推荐类别(Ⅰ/Ⅱa/Ⅱb/Ⅲ) + 证据级别(A/B/C) | evidence-levels.md §2 | "KDIGO推荐" ← 缺年份/级别 |
| 专家共识地位 | 共识全称 + 发布机构 + 年份 + 本品具体地位（一线/二线/提及） | evidence-levels.md §4.3 | "共识推荐" ← 缺名称/机构/年份 |
| 关键结果 | 效应量/率差/HR/OR + 置信区间 + p值 | 研究原文 | — |

### 1.4 锚点判定规则

- **锚点人群**（⭐）：五维中有≥4维达到高等级（证据1a/1b + N≥1000 + 指南Ⅰ或Ⅱa/A级 + 共识一线推荐 + 显著阳性结果）。
- **二批人群**：3维达标（证据1b + 有指南推荐 + 显著结果，但样本量或共识力度不足）。
- **三批/机会人群**：≤2维达标（证据等级低/样本量小/无指南推荐/仅有探索性结果）。
- **证据空白**：该格子无任何研究数据 → 标「待证据补充」，不纳入首批推广。

### 1.5 缺失处理
- 某格查不到数据 → 标 💭「待核实」，不得编造。
- 仅有厂商材料、无独立文献/指南支撑 → 标 💭，不得当 🔍。
- 无头对头研究 → 明确写「暂无头对头证据」，不得暗示等效。

## 2. STP 细分框架

### 2.1 患者细分（Segmentation）
以证据强度为**第一排序维度**，叠加：
- 疾病严重度（轻/中/重/危重）。
- 人口学（年龄/性别/合并症）。
- 行为特征（依从性/就诊习惯/信息获取渠道）。

产出：患者分群表，每群标证据强度等级与市场容量。

### 2.2 目标选择（Targeting）
- 首批目标 = 锚点人群（证据最强 + 市场容量可接受 + 竞争空白）。
- 二批 = 证据次强人群（需补证据或等指南更新）。
- 三批 = 机会人群（待证据补充后评估）。

### 2.3 定位对接（Positioning）
- 每批目标人群对应一条核心信息（见 `promotion-toolkit.md` §2 核心信息矩阵）。
- 锚点人群用核心信息，二批用支持信息降维处理。

## 3. 处方者细分（首处方者画像）

### 3.1 锚点人群的首处方者
谁最先处方这个产品？在什么场景下处方？
- 首处方者画像：科室 / 职称 / 学术活跃度 / 观念立场。
- 处方场景：首诊 / 换药 / 联合用药 / 长期维持。

### 3.2 处方者分层
| 层级 | 定义 | 市场策略 |
|---|---|---|
| KOL（观念领袖） | 全国指南执笔/主委级 | 深度合作：指南/顾问/发声 |
| DOL（区域领袖） | 省级主委/区域带头人 | 区域覆盖：讲者/科室会 |
| MNV（意见处方者） | 高处方量/同行影响大 | 精准触达：拜访/数据传达 |
| 普通处方医生 | 常规处方 | 规模覆盖：DA/学术活动/数字化 |

### 3.3 市场 KOL vs 医学 KOL（边界）
| | 市场 KOL | 医学 KOL |
|---|---|---|
| 导向 | 推广导向（讲者培养/观念领袖/规模传播） | 学术导向（科学交流/临床研究/MSL管理） |
| 管理方 | 市场部 | 医学部 MSL |
| 场景 | 学术会议讲者/DA代言/观念播种 | 科学深谈/IIS合作/医学问询 |
| 协同 | 市场做规模传播，医学做深度科学 | 医学建立科学信任，市场放大影响 |

## 4. 机构细分

锚点人群集中的机构类型：
- 三级综合 / 三级专科 / 二级 / 基层 / 民营。
- 按锚点人群分布密度排序，产出机构覆盖优先级。

## 5. 区域优先级

锚点人群地理分布 × 市场容量 × 增长率：
- 核心区域（锚点人群密集 + 容量大 + 增长快）→ 优先投入。
- 发展区域（容量中等 + 增长潜力）→ 逐步覆盖。
- 维持区域（容量小或饱和）→ 维持性投入。

## 6. 人群优先级矩阵（决策输出）

| | 市场容量大 | 市场容量中 | 市场容量小 |
|---|---|---|---|
| **证据强** | ⭐首批目标（锚点） | 首批/二批 | 二批 |
| **证据中** | 二批（补证据） | 二批/三批 | 三批 |
| **证据弱** | 三批（等证据） | 三批 | 暂不进入 |
| **竞争空白** | 加分项（优先） | 加分项 | 考虑进入 |

### 6.1 每批目标须标
- 进入逻辑（为什么这批优先）。
- 所需证据补充（二批/三批需补什么研究/等什么指南）。
- 预计进入时间（绑定注册里程碑/指南更新节奏）。
- 资源配置建议（人力/物料/活动倾斜比例）。

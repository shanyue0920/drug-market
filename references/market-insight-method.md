# 市场洞察方法论（速查）

> 本文件是 `drug-market` Skill 的 references 之一，供簇A「市场洞察与情报」引用。
> 用途：统一市场洞察的调研框架与数据源标准，避免凭记忆给市场规模与竞品情报。
> last-reviewed: 2026-07-10

## 1. 治疗领域分析框架

### 1.1 市场规模测算
调用 `references/market-methodology.md` §2，强制标注口径：
- 价格口径：终端价（医院/零售）/ 出厂价 / 医保支付价。
- 地域口径：中国 / 全球 / 特定区域。
- 人群口径：确诊患者 / 估测总患病人群。
- 数据源：IQVIA / 米内 / 中商产业 / 公开年报 / 专家访谈。

### 1.2 增长趋势分析
- 历史增速（3-5年 CAGR）。
- 驱动因素：患病率变化 / 诊疗率提升 / 新药上市 / 支付扩容。
- 阻碍因素：集采降价 / 仿制冲击 / 指南降级 / 安全性事件。

### 1.3 患者流漏斗
调用 `references/market-methodology.md` §1，各环节标比例与来源，定位最关键瓶颈环节。

### 1.4 未满足需求（UNMET NEED）四维评估
| 维度 | 评估问题 | 信号 |
|---|---|---|
| 疗效 | 现有疗法有效率/达标率是否足够？ | 达标率<50%→强需求 |
| 安全 | 现有疗法不良反应负担？ | 高停药率→需求 |
| 便利 | 给药频次/途径是否便利？ | 日制剂→依从性痛点 |
| 可及 | 患者能否获得/支付？ | 进院率低/自费高→可及缺口 |

## 2. 竞品监测框架

### 2.1 竞品分类
- **直接竞品**：同靶点/同机制已上市产品。
- **间接竞品**：同适应症、不同机制/通路的产品。
- **替代竞品**：不同治疗方式（如手术 vs 药物、器械 vs 药物）。
- **管线竞品**：在研产品（III期/申报中/即将上市），关注预期上市窗口。

## 3. 处方者洞察框架

### 3.1 处方行为分析
- 处方驱动因素：循证证据 / 指南推荐 / 患者需求 / 同行影响 / 代表拜访 / 学术活动。
- 处方障碍：安全性顾虑 / 价格 / 进院难 / 习惯惯性 / 竞品忠诚。
- 处方场景：首诊处方 / 换药决策 / 联合用药 / 长期维持。

### 3.2 观念分层（ADOPTION CURVE）
| 层 | 占比(估) | 特征 | 市场策略 |
|---|---|---|---|
| 早期采纳者 | 10-15% | 关注新数据/愿意率先尝试 | 上市首波重点覆盖 |
| 早期主流 | 30-35% | 等指南推荐/同行验证 | 二批扩展主力 |
| 晚期主流 | 35-40% | 跟随大流/需多次触达 | 成熟期覆盖 |
| 保守者 | 15-20% | 习惯惯性/抗拒换药 | 维持性触达 |

### 3.3 信息获取渠道
学术会议 / 期刊文献 / 同行交流 / 代表拜访 / 数字平台 / 继续教育。

## 4. 患者洞察框架

### 4.1 患者旅程映射
就诊（哪级医院/哪个科室）→ 确诊（诊断手段/时间延迟）→ 治疗（一线/二线选择）→ 依从（持续用药/脱落原因）→ 达标（疗效评估/方案调整）。

### 4.2 依从性障碍
- 副作用 / 用药复杂 / 费用负担 / 认知不足 / 随访缺失。
- 依从性数据源：处方续方率 / 患者登记 / RWE。

## 5. 政策环境解读（市场视角，非准入执行）

### 5.1 DRG/DIP 影响
- 对处方行为的约束（同类药品费用控制）。
- 对新品进院的阻力（DRG分组未覆盖新药）。
- 市场策略应对：疗效经济学叙事 / 补充RWE / 特殊病例申诉。

### 5.2 VBP（集采）影响
- 集采品种的市场重构（原研 vs 仿制份额变化）。
- 非集采品种的窗口机会（竞品被集采→本品替代机会）。
- 市场策略应对：差异化定位 / 渠道转移（DTP/院外）。

### 5.3 医保目录调整
- 谈判品种的市场准入节奏（谈判→执行→放量曲线）。
- 目录调整周期对上市节奏的影响。

> **边界**：政策解读只做"对市场影响"，医保申报/集采投标/挂网执行归准入 skill。

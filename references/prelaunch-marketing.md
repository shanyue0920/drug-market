# 上市前市场与上市准备（速查）

> 本文件是 `drug-market` Skill 的 references 之一，供簇H「上市前市场与上市准备」引用。
> 用途：统一上市前市场预研与上市准备的框架，确保上市即有策略、有节奏、有弹药。
> last-reviewed: 2026-07-10

## 1. 上市前市场调研（H1）

### 1.1 疾病 Landscape
- 患病率/发病率/诊疗路径/未满足需求（调用 `drug-base/references/market-methodology.md` §1 患者流）。
- 数据源：流行病学文献/指南/登记数据/专家访谈。
- 产出：疾病全景图 + 患者流漏斗 + 未满足需求清单。

### 1.2 竞品管线追踪
- 在研竞品进度（I/II/III期/申报中/即将上市）。
- 预期上市窗口（本品 vs 竞品谁先上市？时间差多大？）。
- 数据源：ClinicalTrials.gov / CDE 药品审评公示 / 企业管线公开信息。

### 1.3 处方者画像预研
- 谁会首批处方？（科室/职称/学术活跃度）。
- 处方驱动因素预判（循证/指南/患者需求/代表拜访）。
- 数据源：专家访谈/处方习惯调研/同类品种上市轨迹。

## 2. 上市前定位预研（H2）

### 2.1 锚点人群预判
- 基于 II/III 期临床数据做证据强度分布图**预研版**。
- 哪个亚组 III 期数据最强 → 预判锚点人群。
- 上市后用真实世界数据验证/修正。

### 2.2 价值主张方向
- 临床价值方向（锚点人群的核心疗效优势）。
- 差异化方向（vs 已上市竞品的预判差异点）。
- 须标注"预研结论，待上市后验证"。

### 2.3 上市后验证计划
- 锚点人群验证：上市后3-6个月看首处方是否集中在预判人群。
- 定位验证：上市后6-12个月看市场份额是否在锚点人群聚集。
- 策略调整：验证不符则回到证据分布图重新定位。

## 3. 上市前 KOL 画像与观念培育（H3）

### 3.1 推广 KOL 画像预研
| 维度 | 预研内容 |
|---|---|
| 学术地位 | 指南执笔/学会任职/发文影响力 |
| 观念立场 | 对本品机制/数据的态度（支持/中立/质疑） |
| 讲者潜力 | 表达能力/学术影响力/适合哪级会议 |
| 竞品关系 | 与竞品KOL的关系/是否已被竞品绑定 |
| 首处方潜力 | 是否会成为首批处方医生 |

### 3.2 观念培育（合规框架内）
- 上市前学术播种：在未获批阶段，通过科学交流传递临床数据（须符合未上市产品科学交流合规要求）。
- 形式：专家咨询会/科学研讨会/数据预发布（合规允许范围内）。
- 目标：上市前建立科学认知，上市后快速转化为处方行为。

### 3.3 与医学协同
| | 市场做（推广KOL） | 医学做（学术KOL） |
|---|---|---|
| 上市前 | 推广画像/观念摸底/讲者预选 | 科学传播（机理/临床设计/数据解读） |
| 上市后 | 讲者培养/观念领袖/规模传播 | 科学深谈/IIS合作/MSL管理 |

## 4. 上市策略与节奏（H4）

### 4.1 Launch Strategy
```
首波（0-6月）：锚点人群深耕 → 建认知/抢首处方/锁定KOL
二批（6-18月）：证据次强人群扩展 → 扩份额/拓人群/渠道下沉
成熟（18月+）：全人群覆盖 → 守份额/防侵蚀/证据深化
```

### 4.2 上市里程碑
| 里程碑 | 市场动作 | 时间窗 |
|---|---|---|
| 批件到手 | 内部启动/上市会筹备 | T+0 |
| 挂网完成 | KOL锁定/首场学术活动 | T+1月 |
| 首批进院 | 核心区域科室会启动 | T+2月 |
| 首张处方 | 媒体传播/案例收集 | T+3月 |
| 爬坡期 | 区域会/数字化上量 | T+3-6月 |
| 稳定期 | 二批人群扩展/份额监测 | T+6-18月 |

## 5. 上市前市场预估（H5）

### 5.1 Peak Forecast（峰值预测）
调用 `drug-base/references/market-methodology.md` §2 自下而上：
```
峰值市场规模 = 锚点人群患者数 × 渗透率峰值 × 年治疗费用
```
- 锚点人群患者数：流行病学数据（标🔍）。
- 渗透率峰值：对标同类品种上市后3-5年渗透率（标💭估算）。
- 年治疗费用：预期定价 × 年用药量（标💭待定价确认）。

### 5.2 渗透曲线（爬坡节奏）
- 对标同类品种上市轨迹（上市后第1/2/3/5年渗透率）。
- 考虑本品差异（证据强度/竞品格局/支付环境）调整。
- 产出：年度渗透率预测曲线 + 敏感性分析（乐观/中性/保守）。

### 5.3 预估质量要求
- 所有预测标🔍（历史对标数据）或💭（估算假设）。
- 关键假设显式列出（渗透率假设/定价假设/竞品假设）。
- 不保证精确，保证逻辑可追溯、假设可挑战。

## 6. 上市准备就绪（H6 · Launch Readiness Checklist）

### 6.1 就绪检查清单
| 维度 | 就绪标准 | 状态 |
|---|---|---|
| 团队 | 市场/销售团队到岗+培训完成 | □ |
| 物料 | DA/折页/视频/数字物料制作完成+MLR审核通过 | □ |
| KOL | 首批KOL锁定+上市会讲者确认 | □ |
| 渠道 | 核心区域经销商确认+首批进院名单锁定 | □ |
| 活动 | 上市会方案确认+首场区域会/科室会排期 | □ |
| 数字化 | 数字平台上线+精准推送系统就绪 | □ |
| 监测 | KPI体系确认+数据源接入+基线建立 | □ |
| 合规 | 推广物料MLR审核+合规培训完成 | □ |

### 6.2 就绪评估
- 全部□打勾 → 上市就绪，可启动上市活动。
- 有未勾选项 → 列风险与缓解措施，评估是否影响上市节奏。

## 7. 定价策略输入（H7 · 定价执行归准入）

### 7.1 价值定位→定价区间
- 价值基准：锚点人群的临床价值 vs 标准疗法。
- 竞品价格基准：同机制/同适应症已上市产品价格。
- 支付意愿基准：医保支付能力/患者自付能力。

### 7.2 定价区间建议
```
定价上限 = 价值溢价 × 标准疗法价格（临床价值支撑的上限）
定价下限 = 成本 + 合理利润（企业可持续的下限）
定价中枢 = 竞品价格基准 × 价值调整系数
```
- 须标注"市场价值定位输入，定价决策与谈判归准入"。
- 与准入药经协同：市场给价值定位，准入做 ICER 与谈判底线。

### 7.3 定价对市场策略的影响
- 高定价策略：锚点人群深耕+DTP/商保渠道+价值叙事。
- 中定价策略：平衡覆盖与价值+医保路径+规模上量。
- 低定价策略：快速渗透+集采应对+规模替代。

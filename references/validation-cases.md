## 案例一：司美格鲁肽（诺和盈®，减重版）· 品牌市场分析

**① 元数据头**
- 分析对象：司美格鲁肽注射液（减重版，商品名诺和盈® / Wegovy）🔍
- 适应症：成人长期体重管理——BMI≥30 的肥胖，或 BMI 27–30 且合并≥1 项体重相关合并症 🔍（注：该标签阈值与《中国成人超重肥胖预防控制指南》BMI≥28 的「肥胖」界值存在差异 ⚠️ 待核实标签原文）
- 分析视角：市场（品牌定位 + 证据驱动细分 + 推广）｜数据截止 2026-07｜证据总览：L1 获批/医保、L2 电商挂网价为主，市场规模为 L4 估算

**② 执行摘要**
1. 诺和盈® 2024-06 获 NMPA 批准、2024-11-17 国内商业化上市，是全球首个长期体重管理 GLP-1 周制剂 🔍。
2. 核心证据 STEP 系列平均减重约 17% 🔍；锚点人群锁定「BMI≥30 或 27–30 合并代谢异常、需长期体重管理」成人。
3. 减重版**不进医保**、患者自费，单支电商价 1000 元以上 ⚠️ 待核实（电商/连锁药房口径，非官方挂网价）。
4. 最大威胁：替尔泊肽（双靶点）疗效更优 + 专利悬崖临近 + 本土仿制排队 💭。
5. 关键不确定性：减重版省级挂网价与自费支付意愿尚未固化 ⚠️ 待核实。

**③ 主体（节选八簇）**
- A 治疗领域：中国超重/肥胖人群基数大、治疗率低=增量蓝海（患者流前段漏损大）💭。
- B 定位：「每周一次、显著减重+心血管获益证据」差异化；定位陈述须落地到合规传播语，禁夸大。
- C 证据强度分布图：减重终点 STEP 1（NCT02970164，NEJM 2021，N≈1961，减重≈15%）🔍 为锚点格；心血管保护 SIGNAL/SELECT 💭 待补强。
- D 推广：on-label 限获批减重适应症；物料标「待 MLR 审核」；竞品卡主打周制剂依从性 + 证据完整性 vs 日制剂/在研仿制。
- E/G：上市爬坡期 KPI 以首处方医生数、锚点人群渗透为先。

**④ 关键结论与行动建议**
- next-step：① 锁定三级医院内分泌/减重门诊为首处方阵地；② 用「筛查常态化」杠杆撬低诊断治疗率蓝海；③ 联防集采/仿制威胁，独占窗口建「循证+指南+RWE+品牌心智」四件套。
- 接口：定价/双通道归准入（H7 给价值定位输入）；推广合规归 MLR。

**⑤ 不确定性·待核实清单**
- ⚠️ 诺和盈® 官方省级挂网价未公开，电商 1396/2126/2706 元等为平台价，口径与时效须以招采平台为准。
- ⚠️ 标签 BMI 阈值（30/27）与中国指南 28 界值差异，需核说明书原文。
- ⬜ SELECT 心血管结局中国亚组数据待补。

**⑥ 信息来源与证据等级**
- NMPA 获批（2024-06）、商业化上市（2024-11-17）🔍 L1；电商价 💭 L6 线索；STEP/NEJM 🔍 L3。
- 免责：仅供战略参考，不构成医疗/投资建议；推广内容须经 MLR 审核。

## 案例二：奥希替尼（泰瑞沙®）· 肿瘤靶向药市场分析

**① 元数据头**
- 分析对象：甲磺酸奥希替尼片（泰瑞沙®，osimertinib）🔍｜第三代 EGFR-TKI
- 适应症：EGFR 突变（19del/L858R）NSCLC 一线、T790M 二线、术后辅助治疗 🔍
- 视角：市场｜截止 2026-07｜证据：L1 批件/医保、L3 FLAURA/ADAURA

**② 执行摘要**
1. 2017 入中国，经 2018/2020/2022 三轮国谈降价，80mg 医保支付标准 165.54 元/片、月费用≈4966 元 🔍。
2. 医保乙类（协议期谈判药，2025 版目录协议至 2026-12-31），覆盖一线/二线/辅助 🔍。
3. FLAURA 一线 mPFS 18.9 vs 10.2 月（HR 0.46）🔍；ADAURA 辅助 DFS HR 0.17（风险降≈83%）🔍 → 锚点人群=EGFR 突变 NSCLC 一线及术后辅助。
4. 威胁：国产仿制（多款过评）价格 2600–3100 元/盒，挤压原研份额 💭。
5. 不确定性：2026 续约是否进一步降价 ⚠️ 待核实。

**③ 主体（节选）**
- C 证据图：FLAURA（Lancet Oncol 2018/NEJM 2020，N=556，HR 0.46）🔍 为锚点；脑转移控制率 70%+ 💭 待原文。
- B/SWOT：S=脑穿透+一线/辅助双证据；W=原研价高；O=辅助适应症放量；T=仿制+续约降价。
- D：核心信息矩阵主打「一线优选+脑保护+辅助治愈窗口」；竞品卡对标一代/二代 TKI 及国产三代仿制。

**④ 关键结论与行动建议**
- 借辅助适应症扩容做增量；用 RWE 说服支付方看「总治疗成本/治愈价值」对抗仿制低价；双通道/DTP 绕单院药事会瓶颈。

**⑤ 不确定性·待核实清单**
- ⚠️ 2026 国谈续约价格是否调整（当前 165.54 元/片为 2025 版标准）。
- ⬜ 国产仿制具体过评企业数与最新中标价待补。

**⑥ 信息来源与证据等级**
- 国家医保局目录/支付标准 🔍 L1；FLAURA/ADAURA 原文 🔍 L3；仿制价为行业报道 💭 L4。

## 案例三：乌司奴单抗（喜达诺®/赛乐信®）· 自免生物药市场分析

**① 元数据头**
- 分析对象：乌司奴单抗注射液（ustekinumab，IL-12/23 p40 抑制剂）🔍
- 原研：喜达诺®（强生/Janssen）；国产类似药：赛乐信®（华东医药/中美华东，2024-10 获批银屑病，国内首款）🔍
- 适应症：中重度斑块状银屑病、克罗恩病（原研 2017 中国获批）🔍｜视角：市场｜截止 2026-07

**② 执行摘要**
1. 原研 2017 中国获批，2021 谈判入医保并续至 2025 版；国产赛乐信® 2024-10 获批、2025-12 入医保，形成「原研+类似药」双供给 🔍。
2. 锚点人群=对传统治疗无效/不耐受的中重度斑块状银屑病（维持期每 3 月一次 45mg，给药频次最低之一）🔍。
3. 价格：原研 45mg 医保价约 4015.74 元/支、90mg 约 6826.76 元/支；赛乐信® 45mg 约 3614 元/支 ⚠️ 待核实（地区/年度口径）。
4. 机会：克罗恩病适应症 2026-05 国产获批，扩人群；威胁：IL-17/IL-23 同类竞品与类似药价格战 💭。
5. 不确定性：赛乐信® 全国实际挂网价与报销比例 ⚠️ 待核实。

**③ 主体（节选）**
- C 证据图：银屑病 III 期（PHOENIX 1/2）🔍 锚点；克罗恩 UNITI/IM-UNITI 💭 待补中国亚组。
- B/D：差异化=「全适应症（皮肤+肠道）+ 低频次」，核心信息矩阵对标 TNF-α 抑制剂与原研心智；竞品卡突出维持期年仅需 4 针的患者价值。

**④ 关键结论与行动建议**
- 借克罗恩新适应症做皮肤科→消化科跨科室扩展；用「低频次=高依从+低年针次成本」对抗价格竞争；杠杆：绑定 IBD 质控/MDT 筛查。

**⑤ 不确定性·待核实清单**
- ⚠️ 原研与赛乐信® 各省实际医保支付价/报销比例，以招采平台与当地医保为准。
- ⬜ 克罗恩病中国真实世界用药渗透率待补。

**⑥ 信息来源与证据等级**
- NMPA 批件/医保目录 🔍 L1；企业公告（华东医药）🔍 L5；价格多为行业报道 💭 L4，标 ⚠️。

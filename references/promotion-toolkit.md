# 推广工具箱（速查）

> 用途：统一推广组合、核心信息矩阵、渠道选择、KOL管理与物料策划的标准。
> last-reviewed: 2026-07-10

## 1. 推广组合策略

### 1.1 推广渠道矩阵
| 渠道 | 定位 | 适用阶段 | 成本量级 |
|---|---|---|---|
| 全国学术会议 | 品牌制高点/权威背书 | 上市期+成长期 | 高 |
| 区域学术会议 | 区域覆盖/观念播种 | 全阶段 | 中 |
| 科室会 | 精准触达/处方转化 | 全阶段 | 低 |

## 2. 核心信息矩阵（围绕锚点人群）

### 2.1 信息层级
| 层级 | 定义 | 证据要求 | 使用场景 |
|---|---|---|---|
| 核心信息 | 锚点人群最强证据对应的主张 | 证据等级1a/1b + 指南推荐 | 所有推广场合的主线 |
| 支持信息 | 非锚点人群的次强证据主张 | 证据等级1b-2a | 特定亚组/竞品对比 |
| 安全信息 | 安全性特征与特殊人群 | RCT安全数据+说明书 | 异议处理/合规需要 |
| 差异信息 | vs竞品的差异化主张 | 头对头或间接比较证据 | 竞品应对/换药决策 |

### 2.2 证据映射
每条信息必须映射到证据强度分布图的对应格子（见 `segmentation-stp.md` §1）：
```
核心信息1 → 适应症A × 亚组X × 终点M → 证据1a, N=10142, ESC Ⅱa/A
支持信息1 → 适应症A × 亚组Y × 终点N → 证据1b, N=3000, ADA B级
```
- 无法映射到证据格子的信息不得用于推广。
- off-label 信息（超获批适应症）严禁用于推广渠道，只限 MSL 科学交流。

### 2.3 信息传递公式
```
[核心信息] = [锚点人群] + [关键终点结果] + [证据等级] + [指南地位]
```
示例：「对于合并心血管高危的2型糖尿病患者，本品显著降低MACE 14%（HR 0.86），证据等级1a，ESC指南Ⅱa类A级推荐。」

## 3. 推广节奏与里程碑

### 3.1 上市首波（锚点人群深耕）
- 时间窗：批件后0-6个月。
- 目标：建认知、抢首处方、锁定KOL。
- 动作：上市会 → KOL锁定 → 核心区域科室会 → 首批目标机构覆盖。

### 3.2 二批扩展（证据次强人群）
- 时间窗：6-18个月。
- 目标：扩份额、拓人群、渠道下沉。
- 动作：区域会 → 二批人群信息适配 → 区域覆盖扩展 → 数字化上量。

### 3.3 成熟期全人群
- 时间窗：18个月+。
- 目标：守份额、防侵蚀、证据深化。
- 动作：防御性推广 → 竞品应对 → RWE补充 → 新适应症预热。

### 3.4 里程碑绑定
- 注册里程碑（调用 drug-reg 接口）：批件→挂网→进院→首张处方。
- 指南更新节奏：指南升级 → 核心信息迭代 → 推广话术更新。
- 竞品动态：竞品上市/集采/安全事件 → 策略调整窗口。

## 4. 多渠道/数字化营销（MCM）

### 4.1 全渠道触达
- 线下（会议/拜访/科室会）+ 线上（数字平台/直播/微信/精准推送）。
- 渠道协同：线下建信任，线上做规模，数据驱动精准触达。

### 4.2 数字化工具
- 数字代表（e-detailing）/ 在线学术平台 / 微信公众号 / 视频号 / 精准推送系统。
- 数据闭环：触达→行为→处方→反馈→优化。

### 4.3 合规边界
- 数字化推广内容同样须 on-label + 待MLR审核。
- 患者端内容须区分处方药/OTC，处方药不得直接面向消费者推广（中国法规）。

## 5. 学术会议策划

### 5.1 会议层级
| 层级 | 类型 | 目标 | 内容 |
|---|---|---|---|
| 全国会 | 卫星会/专题会 | 品牌制高点 | 锚点人群最新数据/KOL发声 |
| 区域会 | 专场/午餐会 | 区域覆盖 | 核心信息+本地化案例 |
| 科室会 | 科室内分享 | 处方转化 | 锚点人群+实操指导 |

### 5.2 策划要素
- 讲者匹配（KOL分层×会议层级）。
- 内容框架（核心信息→证据→案例→讨论）。
- 互动设计（Q&A/病例讨论/投票）。

## 6. 推广 KOL 管理

### 6.1 推广 KOL vs 医学 KOL
详见 `segmentation-stp.md` §3.3。推广 KOL 偏市场导向，医学 KOL 偏学术导向，须协同。

### 6.2 讲者培养路径
```
科学理解（懂证据）→ 独立宣讲（能讲清）→ 观念领袖（信且推）→ 品牌代言（公开发声）
```
每阶段须有培养动作与评估标准。

### 6.3 KOL 画像模板
- 学术方向 / 发文记录 / 学会任职 / 临床实践特点。
- 观念立场（支持/中立/质疑）/ 竞品关系 / 讲者能力。
- 推广价值：适合哪级会议 / 哪个锚点人群领域 / 发声影响力。

## 7. DA 物料策划

### 7.1 物料类型
| 类型 | 用途 | 核心内容 |
|---|---|---|
| DA（详情页） | 代表拜访核心工具 | 核心信息+证据图表+竞品对比 |
| 折页 | 快速概览 | 一句话定位+核心数据 |
| 视频 | 会议/线上 | 锚点人群故事+KOL发声 |
| 数字物料 | 数字化推广 | 交互式DA/微视频/H5 |

### 7.2 内容框架
- 每页/每屏聚焦一条信息（核心 or 支持）。
- 证据映射到分布图格子，标注来源与等级。
- 竞品对比须公平、有据、不贬低。
- **所有物料标注"待MLR审核"**，不替代合规审查。

# 品牌定位框架（速查）

> 本文件是 `drug-market` Skill 的 references 之一，供簇B「品牌战略与定位」引用。
> 用途：统一品牌定位的框架与产出标准，确保定位基于证据、可落地传播。
> last-reviewed: 2026-07-10

## 1. 品牌定位方法论

### 1.1 定位三角
```
        临床价值（证据锚点）
           / \
          /   \
   差异化主张   目标人群
          \   /
           \ /
        竞争空白（对手做不到/没做好的）
```
定位 = 临床价值 ∩ 目标人群 ∩ 竞争空白。三者交集才是有效定位。

### 1.2 差异化定位陈述模板
调用 `references/market-methodology.md` §4.2：
```
针对 [锚点人群/适应症]，
[产品名] 是 [类别]，
其核心差异在于 [关键临床/可及性优势]，
相比 [主要竞品] 能为 [医生/患者/支付方] 带来 [具体价值]。
```
- 须可落地到一句传播语。
- 须基于证据强度分布图（簇C），不得脱离证据定位。

### 1.3 定位验证三问
1. **真实吗**？——每条主张能否映射到证据强度分布图的格子？
2. **差异吗**？——竞品能不能说同样的话？如果能，换。
3. **重要吗**？——医生/患者/支付方在乎这个差异吗？

## 2. 品牌架构管理

### 2.1 单适应症 vs 多适应症
- 单适应症：聚焦定位，资源集中。
- 多适应症：需管理适应症优先级与品牌一致性。
  - 优先级排序：证据强度 × 市场容量 × 竞争空白（调用簇C C6矩阵）。
  - 一致性：核心品牌主张不变，适应症层信息各有侧重。

### 2.2 产品线组合
- 主力产品 / 成长产品 / 防御产品 / 种子产品。
- 资源分配：主力保份额、成长抢增量、防御挡竞品、种子育未来。

## 3. 生命周期管理（PLC）

### 3.1 各阶段策略
| 阶段 | 核心目标 | 策略重点 |
|---|---|---|
| 上市期 | 建认知、抢首处方 | 锚点人群深耕、KOL锁定、上市活动 |
| 成长期 | 扩份额、拓人群 | 二批人群扩展、渠道下沉、数字化上量 |
| 成熟期 | 守份额、防侵蚀 | 证据深化、差异化巩固、防御竞品 |
| 衰退期 | 延生命周期、有序退出 | 新适应症/剂型延伸、专利悬崖应对 |

### 3.2 专利悬崖应对
- 独占窗口内建「循证+指南+RWE+品牌心智」四件套。
- 布局差异化剂型/复方/新适应症延长生命周期。
- 仿制上市前6-12个月启动防御策略。

### 3.3 新适应症拓展
- 拓展逻辑：证据强度分布图中"次强格子"→ 补临床试验 → 新适应症申报（调用 drug-reg 接口）→ 上市后推广扩展。
- 每个新适应症须重新走证据驱动细分流程（簇C）。

## 5. 价值证据包（Value Dossier）

### 5.1 三维价值主张

| 维度 | 内容 | 证据来源 |
|---|---|---|
| 临床价值 | 锚点人群疗效/安全/便利 | 循证三件套（调用 evidence-levels.md） |
| 经济价值 | 成本效果/预算影响/总治疗成本 | 药经数据（准入提供，市场整合） |
| 患者价值 | 依从性/生活质量/便利性 | RWE/患者报告结局 |

### 5.2 证据强度分布图（核心组件）
- 按「适应症×亚组×关键终点」三维交叉建表。
- 每格标注：证据等级 / 样本量 / 指南推荐 / 专家共识 / 关键结果。
- 标出锚点人群（证据最强格子）。
- 详见 `segmentation-stp.md` §1。

### 5.3 价值证据包用途
- 内部：品牌策略依据、推广信息溯源。
- 准入：医保谈判/药事会价值档案（调用准入接口）。
- 医学：与医学核心信息包对齐（医学给科学主张，市场做推广转化）。
- 销售：销售工具包的证据基础。
